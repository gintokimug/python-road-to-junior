2. **Python**
    Что касается директорий,можно произвести import os, после чего открывается функционал действий ,по типу 
    - print(os.getcwd()) даёт путь директории в которой ты находишься
    - os.chdir('C:\\Python33')  смена директории, скорее всего необходимо для перетаскивания файлов в больших проектах.
    - os.listdir()  получаешь весь  список subdirectories, и файлом текущей директории в котороой ты находишься(наверно пригодится для нахождения ошибок, лишних файлов или проверок,)
    - os.mkdir('test') создание новой директории, если не пропишешь пуТЬ , то создатся в текущей рабочей
    - os.remove("myfile.txt") для удаления файла 

3. **CSV**
    Узнал, что это формат файла для чтения табличных данных. ПРименяется в data science , очень нужен для огромного кол-ва данных , импортируется модулем  csv 

    import csv

`with open('people.csv', 'r') as file:`
    reader = csv.reader(file)

    for row in reader:
        print(row)

        Вот такая штука нужна для чтения, НО, есть ещё более удобная, представляющая данные в виде словарей:
    `import csv`
`with open('people.csv', 'r') as file:`
    `csv_file = csv.DictReader(file)`
    `for row in csv_file:`
        `print(row)`
        
Здесь мы использовали csv.DictReader(file), чтобы первая строка CSV-файла считалась заголовками столбцов, а каждая последующая строка — записью данных.
     Ещё есть штука для написания файлом csv

with open('protagonist.csv', 'w', newline='') as file:
