Конспект: Классы и объекты в Python (Атрибуты)

1.  *Основы определения*

  - Создание класса: Используется ключевое слово class. По стандарту PEP 8 имена пишутся с заглавной буквы (например, class Point:).
  - Атрибуты класса: Переменные, объявленные внутри класса. Они образуют пространство имен класса и доступны всем его экземплярам.

2.  *Экземпляры (объекты)*

  - Создание объекта: Вызывается как функция: a = Point().
  - Проверка типа: Функции type(a) или isinstance(a, Point).

3.  *Работа с атрибутами*

  - Поиск атрибута: Сначала Python ищет атрибут в самом объекте (локально), а если не находит — обращается к классу.
  - Локальные атрибуты: Если присвоить значение a.color = 'green', у объекта a появится свой атрибут, который "перекроет" атрибут класса для этого конкретного объекта.
  - Служебный атрибут dict: Позволяет увидеть все текущие атрибуты объекта или класса в виде словаря.

4.  *Встроенные функции*

  - setattr(obj, 'name', value) — установить значение атрибута.
  - getattr(obj, 'name', default) — получить значение (можно задать значение по умолчанию, чтобы избежать ошибки).
  - hasattr(obj, 'name') — проверить наличие атрибута.
  - delattr(obj, 'name') или del obj.name — удалить атрибут.

5.  Документирование

  - Docstrings: Строка в начале класса попадает в служебный атрибут doc.
